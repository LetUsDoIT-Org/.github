# LetUsDoIT

We build practical apps that solve everyday problems — from garden care to food planning to mileage tracking.

---

## Projects

### 🌱 Havemakker

A mobile app for Danish gardeners to track plants, manage care schedules, and get seasonal reminders. Climate-aware scheduling based on frost dates, with hand-curated Danish plant data.

| Repository | Description |
|---|---|
| [havemakker](https://github.com/LetUsDoIT-Org/havemakker) | Planning, epics, and shared project docs |
| [havemakker_frontend](https://github.com/LetUsDoIT-Org/havemakker_frontend) | Expo React Native app |
| [havemakker_web](https://github.com/LetUsDoIT-Org/havemakker_web) | Marketing website |

**Tech:** React Native (Expo) · Supabase · TypeScript · Gluestack UI · NativeWind
